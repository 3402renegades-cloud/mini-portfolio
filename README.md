# ミニポートフォリオ

React + TypeScript + Vite で作ったミニアプリ集です。ルーティングで各アプリを行き来でき、日常のちょっとした課題（時間管理、文章作成の集中力アップ、気分転換など）を解決するために自作しました。

## デモ
- GitHub Pages（予定）: https://your-username.github.io/mini-portfolio/
- スクリーンショット: 公開後に追加します

## アプリ一覧と意図
- Timer: 3秒/10秒/30秒などのプリセットと Start / Pause / Reset を備えたカウントダウン。
- Typing Game: カウントダウン開始、ミスタイプ数、ベストタイム表示付き。誰でも遊びやすい練習用ゲーム。
- Omikuji: ワンクリックでおみくじを引く簡易アプリ。履歴表示や演出の追加を見据えた構成です。
- Slot Game: シンプルなスロット演出と判定ロジックを実装。

## 使い方
```bash
npm install
npm run dev
```
- ローカル起動後、左上のナビゲーションから各アプリへ移動できます。

## 品質チェック
学習と併せて基本的な品質管理を行っています。
```bash
npm run lint   # コード整形・静的解析
npm run build  # TypeScript ビルド + Vite 本番ビルド
```

## 学びとアピールポイント
- 企画→設計→実装→検証→公開 まで一通り経験するために、複数のミニアプリを1つのポートフォリオとして統合しました。
- UI 文言や操作説明を充実させ、非エンジニアにも伝わるように工夫しています。
- 状態管理（カウントダウン、ベストスコア更新、入力ミスカウント）やイベント制御（キー入力、クリック）の実装例として示しています。
