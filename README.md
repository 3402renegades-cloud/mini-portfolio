# ミニポートフォリオ

React + TypeScript + Vite で作ったミニアプリ集（ポートフォリオ）です。  
学習用に作った4つのアプリを1つに統合し、ルーティングで切り替えられるようにしています。

## アプリ
- **タイマー**：プリセット切替 / Start / Pause / Reset 付きカウントダウンタイマー
- **タイピング**：カウントダウン開始、ミスタイプ数、ベストタイム表示
- **おみくじ**：おみくじ表示（※実装した機能：例）履歴 / 演出 / 連続で引けない制御 など
- **スロット**：スロットゲーム（※実装した機能：例）判定 / 演出 / ボタン制御 など

## 技術スタック
- React
- TypeScript
- Vite
- React Router（導入している場合）
