# ミニポートフォリオ

React + TypeScript + Vite で作ったミニアプリ集です。ルーティングで各アプリを行き来できます。

## アプリ一覧
- Timer: 3秒/10秒/30秒などのプリセットと Start / Pause / Reset を備えたカウントダウン。
- Typing Game: カウントダウン開始、ミスタイプ数、ベストタイム表示付きのタイピングゲーム。
- Omikuji: ワンクリックで結果を表示するおみくじアプリ。
- Slot Game: シンプルなスロット演出と判定ロジックを実装。

## 使い方
```bash
npm install
npm run dev
