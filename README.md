# mini-portfolio

React + TypeScript + Vite で作ったミニアプリ集（ポートフォリオ）です。  
学習用に作った4つのアプリを1つに統合し、ルーティングで切り替えられるようにしています。

## Apps
- **Timer**：プリセット切替 / Start / Pause / Reset 付きカウントダウンタイマー
- **Typing**：カウントダウン開始、ミスタイプ数、ベストタイム表示
- **Omikuji**：おみくじ表示（演出/履歴など、実装している機能を記載）
- **Slot**：スロットゲーム（演出/判定など、実装している機能を記載）
